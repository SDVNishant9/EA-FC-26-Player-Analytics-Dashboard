# ⚽ EA FC 26 Player Analytics Dashboard

## 📌 Overview
This project is an interactive Power BI dashboard that analyzes player data from EA FC 26.  
It helps in comparing player performance, understanding key metrics, and making data-driven decisions for team building.

---

## 🎯 Objectives
- Analyze player attributes and ratings
- Compare players based on performance metrics
- Identify top performers across positions
- Provide insights for squad selection and scouting

---

## 📊 Features
- Dynamic filters (position, rating, nationality, club)
- KPI cards (overall rating, pace, shooting, passing, etc.)
- Player comparison visuals
- Trend and distribution charts
- Interactive and user-friendly UI

---

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel / CSV Dataset
- Data Cleaning & Transformation (Power Query)
